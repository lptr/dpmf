    kilép a játékból
  **/

/nextlevel

  **/

/step
  /**
    léptet egy kört. ez után hajtódnak végre a kiadott parancsok
    (kivéve az információkérést.)
  **/

/exitgame <name>
  /**
    kilép a játékból, a játékos nevét <name>-re állítja,
    visszalép a highscore táblába
  **/

/** innen kezdve a tesztelést megkönnyítõ, parancsok következnek **/
/******************************************************************/

/remark <name>
  /**
    megjegyzés. a kimeneten /* name */  formában. a szinkronizálásban segít
  **/

/info [all|pacman|monster|bomb|crystal|bonus]
  /**
    all: all below
    pacman: pozíció, irány, objnum
    monster: pozíció, irány, objnum
    bomb: pozíció, élettartam, objnum
    bonus: pozíció, élettartam, tartalom, objnum
    crystal: pozíció, objnum
  **/

/show [maze|info] 
  /**
    maze: szöveges labirintus megjelenítés
    info: score, lives, bombs, time kiírása
  **/

/hack load <filename>
  /**
    betölt egy pályát, ha a pálya befejezõdött, kilép
  **/

/hack score <score>
/hack lives <lives>
/hack time <time>
  /**
    általános hackelés
  **/

/hack pacman kill
/hack pacman set <x> <y> <direction> <bombs>
  /**
    pacman buzerálása
  **/

/hack monster create [clever|dumb] <x> <y> <direction>
/hack monster <objnum> [kill|reborn|remove]
/hack monster <objnum> set <x> <y> <direction>
/hack monster <objnum> [auto|manual]
/hack monster <objnum> <direction>
  /**
    monster átszabása
  **/

/hack bomb create <x> <y> <timetolive>
/hack bomb <objnum> remove
/hack bomb <objnum> set <x> <y> <timetolive>

  /**
    bombával kapcsolatos hack-ek
  **/

/hack bonus [auto|manual]

    bonus elhelyezése automatikus vagy manuális
  **/

  /**
    bonus izélgetése
  **/

    kristály bizergálása
  **/

    az (<fx>,<fy>) mezõ adott falát bekapcsolja/kikapcsolja.
