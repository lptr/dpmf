DPMF PacMan

A project

A DPMF PacMan egy egyetemi projectnek indult egy egy-féléves tárgy - projectlabor - keretében, a Budapesti Mûszaki Egyetem Villamosmérnöki Karának Informatika szakán. Aztán ez lett belõle.

A feladat egy "Kisgömböc" címû játékprogram megírása volt, amelyben bemutathattuk, hogy mennyit tanultunk elõzõ félévben a Programozás technológiája és az Objektum-orientált programozás területén. Hát mi ennyit.

A kész programnak Java 2 Runtime Environment 1.3 alatt kellett futnia. A fejlesztés a teljes félévet igénybe vette, minden héten meg volt, hogy mit kell leadni. A pontos project-definíció itt található.

 

Amit megvalósítottunk

Van dokumentációnk, dögivel. Tizenöt hét alatt összejön egy kevés, ha az ember komolyan dolgozik. És azt hiszem, mi tényleg komolyan vettük a feladatot.

Van most egy PacMan-ünk, ami fut Java 1.3 alatt bármilyen gépen - na jó, azért egy PII-450 nem árt legalább. Finoman mozog, animált, van benne Highscore, és ami a legfontosabb: továbbfejleszthetõ.

Futtatás

A program egy .jar file-ban található. Elindítani a

java -jar PacMan.jar

parancsal lehet, amennyiben valamilyen Java 2 Runtime Environment 1.3 elérhetõ.

 

Letöltés

A program szabadon terjeszthetõ, továbbfejleszthetõ. Tölts le bátran bármit:

  • A futtatható DPMF PacMan program az alap GrassWorld pályával (Zip, 847 KByte)

  • A program forráskódja (Zip, 196 KByte)
    (Ha csak ezt töltöd le, a program nem fog futni, mivel kellenek neki pályák is. Töltsd le hozzá a futtatható verziót, abban van egy. :)

  • A project teljes dokumentációja (PDF Zip, 694 KByte)

  • A project teljes dokumentációja (Microsoft Word RTF Zip, 1 324 KByte)

  • Nemsokára lesznek új pályák is, plusz egy pályaszerkesztõ is - természetesen Javaban.

A futtatáshoz szükség van a Java 2 Runtinme Environment 1.3-ra (vagy inkább akkor már az 1.3.1-re, vagy újabbra)

  • Sun Java 2 JRE 1.3.1

A fordításhoz meg persze Java 2 SDK 1.3 szükséges. Mi ezt használtuk:

  • Sun Java 2 JDK 1.3.1
